## ✨ Features

- 🛒 **Shopping Cart** — Add, remove, and update product quantities with live total
- 🎨 **Responsive Design** — Works seamlessly on desktop, tablet, and mobile
- 🖼️ **Product Listings** — Clean product cards with images, descriptions, and pricing
- ⚡ **Pure Vanilla JS** — Zero external dependencies, fast page loads
- 🎯 **Clean UI/UX** — Intuitive user experience with visual feedback

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/adityapanda045/ecommerce-website.git
cd ecommerce-website

# No build step needed — just open in browser!
# Double-click index.html or serve it:
npx serve .
```

## 📸 What It Does

1. **Browse Products** — View all products in a responsive grid
2. **Add to Cart** — Click to add items with quantity tracking
3. **Cart Management** — Update quantities, remove items, see running total
4. **Checkout UI** — Clear order summary (frontend only)
